## В ходе данной практики доработал:

1. Приложение "Калькулятор" 
   * работа с форками
   * отработка навыка работы с Math и численными/строковыми переменными
2. Приложение "Угадайка":
   * Math и численными/строковыми переменными
   * if-else, работа с массивами, тернарными операторами, switch
3. Приложение для генерации карточек пользователей
   * работа с объектами, их свойствами и методами
   * расширение JSON и работа с ним
   * особенности работы с объектом Date
